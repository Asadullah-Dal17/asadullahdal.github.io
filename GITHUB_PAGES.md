# GitHub Pages Deployment

This site is ready for GitHub Pages!

## How to enable Pages

1. Go to your repository on GitHub.
2. Click on **Settings** > **Pages** (or "Pages" in the sidebar).
3. Under "Build and deployment", set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or `master` if that's your default)
   - **Folder**: `/ (root)`
4. Save. Your site will be published at:
   https://asadullah-dal17.github.io/portfolio-site/

## Notes
- Your main HTML file must be named `index.html` and be in the root of the repository (which it is).
- All asset paths (CSS, images, etc.) are relative and will work on GitHub Pages.
- If you use a custom domain, set it in the Pages settings.

---

For advanced customizations, see the [GitHub Pages documentation](https://docs.github.com/en/pages).
