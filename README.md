# My Portfolio Site

This is the personal portfolio website of Asadullah Dal, a Computer Vision Engineer and AI Consultant. The site showcases my professional experience, technical skills, featured projects, and ways to connect with me.

## Features
- Modern, responsive design
- Dark mode and light mode toggle
- Animated typing effect for professional titles
- Project filtering and interactive project cards
- Timeline of professional experience
- Contact form and social links
- Hosted on GitHub Pages for free and easy access

## Technologies Used
- HTML5 & CSS3 (custom, no frameworks)
- JavaScript (vanilla, for interactivity)
- Font Awesome for icons
- Google Fonts for typography

## Hosting
This site is hosted using [GitHub Pages](https://pages.github.com/), making it fast, reliable, and free to access at any time.

## How to Use
- Visit the live site at: `https://asadullah-dal17.github.io/asadullahdal.github.io/`
- Browse through the sections to learn more about my background, skills, and projects.
- Use the contact form or social links to get in touch.
